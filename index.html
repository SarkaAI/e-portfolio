Modules
LAUNCH INTO COMPUTING
This module provided a foundation in core computing principles and software development skills essential for further study in artificial intelligence. Key learning outcomes included developing proficiency in Python programming, understanding software architecture and data handling, and applying inclusive and ethical design principles. Emphasis was placed on recognising and reducing algorithmic bias, ensuring accessibility, and exploring techniques such as k-anonymity for privacy-preserving data analysis. Learners also gained practical experience in using tools to evaluate maintainability, scalability, and security of code. The module culminated in a final assignment involving the design and implementation of utility applications in both Python and JavaScript to compare development models and language features.
