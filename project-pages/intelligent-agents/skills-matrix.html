Professional Skills Matrix and Action Plan
My self-assessment against the skills framework and my plan for ongoing development following the Intelligent Agents module.
Overview
This module developed a distinct set of technical and professional competencies. Technically, I progressed from understanding agent architectures in theory to implementing a complete multi-agent system: designing with the Tropos methodology and BDI concepts, orchestrating agents on LangGraph, integrating academic APIs with a vector store for grounded retrieval, and running fully local models via Ollama. I built and tested safety mechanisms including bounded replanning loops, deterministic confidence scoring, and provenance-based explainability, supported by a test suite of 147 passing tests.
Professionally, the module strengthened my critical evaluation skills, particularly the ability to audit my own claims. Responding to tutor feedback that exposed factual errors in our group design, I learned to verify framework semantics before committing to them and to document deviations transparently. Collaboration in a virtual team improved my coordination and written communication, while the discussion forums developed my ability to argue from literature on ethics, governance, and the security of autonomous systems, connecting the module to my cybersecurity background.
My action plan has three priorities. First, deepen formal evaluation of LLM-based agents, including groundedness and calibration benchmarks, so that my confidence claims rest on measurement rather than counting. Second, specialise in the security of agent communication, from classical agent communication languages to modern protocols such as MCP, where I can combine this module with my professional focus. Third, extend from single bounded workflows to genuinely multi-agent deployments, studying emergent behaviour and the governance frameworks, such as the EU AI Act and NIST AI RMF, that responsible deployment requires.
