Machine Learning
Learning practical machine learning algorithms, model training, and deployment strategies for intelligent systems.
Learning Outcomes
This module covered the foundations and practical applications of machine learning, including supervised and unsupervised learning techniques, model evaluation, feature engineering, and the principles behind neural networks and deep learning architectures.
Emphasis was placed on building, training, and evaluating models using real-world datasets, understanding the trade-offs between model complexity and generalisation, and critically considering the ethical implications of deploying ML systems in production environments.
