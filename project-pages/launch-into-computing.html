Launch into Computing
A foundation in core computing principles and software development skills essential for further study in artificial intelligence.
Learning Outcomes
This module provided a foundation in core computing principles and software development skills essential for further study in artificial intelligence. Key learning outcomes included developing proficiency in Python programming, understanding software architecture and data handling, and applying inclusive and ethical design principles.
Emphasis was placed on recognising and reducing algorithmic bias, ensuring accessibility, and exploring techniques such as k-anonymity for privacy-preserving data analysis. Learners also gained practical experience in using tools to evaluate maintainability, scalability, and security of code.
